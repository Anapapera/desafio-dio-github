                                                                       Git e GitHub
	O Git foiu criado em 2005, é um sistema de versionamento de código distribuído, criado pelo Linus Torvalds. Criador por trás do Linux, e o Linus estava enfrentando um problema complicado. Software não é feito sozinho.
Não é um trabalho de um homem só. É feito de forma colaborativa, e com o Linux não foi diferente. O Linus escreveu o Linux da maneira colaborativa, às vezes no mesmo arquivo, às vezes alterando a mesma linha complexo suficiente para que suportasse pessoas do mundo inteirotrabalhando,
trabalho que ele estava executando ali e que fosse, então, ele precisava ter um sistema que "versionasse", criasse versões, ele tinha uma equipe, e essa equipe estava executando distribuída no mundo inteiro.
Software não é feito, se não for colaborativo. E ele é feito de forma colaborativa.
	Então, é isso que o Git soluciona, eçe é um sodtware de versionamento de código.
	Hoje em dia nós temos o GitHub, uma empresa da Microsoft, muito bem estabelecida no mercado, o Git e o GitHub são muito forte no mercado hoje em dia, eles aparecem geralmente juntos. São duas tecnologias totalmente diferebtes e complementares.
